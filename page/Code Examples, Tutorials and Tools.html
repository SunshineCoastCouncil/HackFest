Code Examples, Tutorials and Tools

To help you rapidly build a prototype for your awesome idea we have collected some handy code examples and tutorials here.

And if you have any trouble following any of them we will have helpers available throughout the event to help.

Working with ArcGIS

ArcGIS is a full featured GIS system. But if you haven't used it before it can be quite daunting. We have put together a few examples of simple ways you can get, and use, data from it.

Two of the ArcGIS services you might be using during the event are http://gisservices.scc.qld.gov.au/arcgis/rest/services and http://cloud.locate.scc.qld.gov.au/arcgis/rest/services

Check out our ArcGIS Examples.

Working with GeoJSON

GeoJSON is becoming a very popular format for working with geospatial information. To make it easier for you to create your own data mash ups you can convert GeoJSON to CSV for editing in Excel.

Check out convertcsv.com to convert GeoJSON to CSV or convert CSV back to GeoJSON.

Displaying a Map

Leaflet is an open-source, lightweight javascript library for creating interactive, mobile friendly maps. They also have some great code examples to get you going quickly.

Check out the Leaflet Tutorials, especially the Using GeoJSON with Leaflet tutorial.

If Leaflet is too much outside your comfort zone you can even build a quick proof of concept with Excel.

Check out our Excel examples.
