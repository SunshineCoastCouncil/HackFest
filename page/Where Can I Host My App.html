Where Can I Host My App?

It isn't a requirement of the competition to have your app available on the internet, but it is certainly a plus.

There are a heap of different options you can use. These are a few of the options available. But feel free to use any others you like.

Microsoft Azure

Azure has an introductory offer with free included credit for the first month. After the initial period you can continue using free options.

Get started with Azure
(requires a credit/debit card to verify your identity)

Amazon Web Services

Amazon Web Services has an introductory free tier for the first year. In addition to that, participants will be receiving AWS credit worth US $50. Once you've received your code at the event you can redeem your credit.

Getting Started with AWS
(requires a credit/debit card to subscribe)

GitHub Pages

If your app consists of static files you can use GitHub Pages to host it for free, no credit/debit card required.
